Wrongly minified CSS.

When [something] strips quoting, it does not escape the plus (+) character properly, which makes the next paragraph use Comic Sans MS or fantasy. Otherwise, it would be in M+ 1mn, if installed, or another monospace font.

THIS SHOULD NOT BE COMIC SANS.

The webpack.config.js file can be modified to disable one of the minified, or both, and see that the CSS is then valid.
